变量

变量的作用

  • 变量是用来保存数据的一个标识符

  • 变量的存储位置

    • 基本类型：保存在栈中
    • 引用类型：引用类型的地址保存在栈中，值保存在堆中

  • 变量的声明

    • 正常的声明：var name = value; var name1, name2, name3;
    • 不正常的声明：name = value; var a=b=c=10;

  • 变量命名规则

    • 以字母、下划线和$ 开头
    • 可以使用汉字，但是非常不推荐
    • 不能使用关键字和保留字
    • 严格区分大小写

数据类型

基本类型

  • string 字符串：以引号包括起来的都是字符串类型

  • number 数字：整数，小数

  • boolean 布尔：只有两个值，true 和 false

  • undefined 未定义：值是undefined, 类型也是undefined

  • null ：null可以当作基本类型，也可以当作引用类型。值本身是一个基本类型，但是通常用来表示引用类型的无效返回值

引用类型

  • Object

  • Function

  • Array

  • String
  • Number
  • Boolean
  • Math

typeof 关键字

用来查看变量的数据类型。可以查看基本类型和函数的类型，其他引用类型返回的结果都是‘function’，不准确。

返回值：字符串

用法：typeof 变量名 | typeof(变量名)

引用类型判断方式

  • Object.prototype.toString.call();

  • 变量 instanceof 类型

  • 变量.isArray()

类型转换

转换为 *number*类型

  • 强制转换

    • Number(); 可以将纯字符串转换为number
    • parseInt(); 将字符串数字部分的整数部分保留。必须以数字开头的字符串
    • parseFloat(); 可以保留整数及小数位。同样必须以数字开头
    • 注意：如果不是合法的字符串，会返回NaN

  • 隐式转换

    • 通过减、乘、除、取余运算在不改变结果值得前提下进行隐士转换
    • 例如：var a = ‘123’; a = a - 0;
转换为 *string*类型

  • 强制转换

    • String(); 通用方法。内部也是调用 toString() 方法实现的
    • 变量名.toStirng(); null和undefined没有toString方法。

  • 隐式转换

    • 任意类型+空字符串。这样可以在不影响结果值得前提下将其转换成string类型
转换成*boolean*类型

  • 强制转换

    • Boolean(); 将任意类型转换后返回 true 或 false

  • 隐式转换

    • !!
    • 第一个逻辑非操作会基于无论什么操作数返回一个布尔值,第二个逻辑非操作则对该布尔值求反

基本类型和引用类型的区别

  • 存储位置和保存方式不同

  • 复制方式不同

    • 基本类型复制：直接将值复制给新的变量
    • 引用类型复制：复制指针给新的变量，两个指针指向同一个堆中的内容

变量作用域

  • 全局作用域：在script标签内部，函数外部。全局变量都可以被访问，包括函数内部

  • 局部作用域：函数内部。局部变量只能在其函数内部使用，如果外部想访问，只能有函数的return将其返回。

  • 注意：Javascript中，只要函数可以分割作用域，也就是只有声明在函数内部的变量才是局部变量
