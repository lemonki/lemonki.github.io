Javascript 基础知识

  • 什么是Javascript?

    • Javascript 是一种运行时边解析边执行的客户端脚本语言。
    • Javascript 是弱类型语言。比如一个变量初始化值是number类型，可以改成string类型。不推荐这样做。

  • 为什么要有Javascript？

    • 最初目的：判断和验证表单
    • 现在的意义：特效、移动端、异步和服务器交互、服务器开发(nodejs)

  • 组成

    • ECMAScript: Javascript的语法标准
    • DOM：操作网页上元素的API
    • BOM：操作浏览器上元素的API

  • Javascript 引用方式

    • 内嵌式：可以写在HTML文档的任意部位。推荐写在body或head标签下
    • 外联式：通过script标签的src属性引用。`<script src=”test.js”><\/script>`
    • 注意：写在引用js文件的script标签内的代码不会执行
