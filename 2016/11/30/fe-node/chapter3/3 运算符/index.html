运算符

  • 算数运算符 + - * \/ %

  • 一元运算符 ++ –

    • 运算符前置：先自增或自减再参与运算
    • 运算符后置：先参与运算在自增或自减

  • 逻辑运算符 && || !

    • && ： 两个条件都为true是才为true
    • || ： 只要有一个为true就是true
    • ! ：取反

  • 比较运算符 > < >= <=

  • 赋值运算符 = += -= *= \/=

  • 相等运算符

    • == != 只判断值是否相等或不等。比较时会先进行隐式转换
    • === !== 同时判断类型和值是否相等或不等

  • 三元(三目)运算符

    • 语法：布尔表达式 ？为真时执行此代码 ：为假时执行此代码
    • 作用：为了替代简单的if else
    • 3个数的运算

运算符的优先级

  • 优先级从高到底

  • () 优先级最高

  • 一元运算符 ++ – !

  • 算数运算符 先* \/ % 后 + -

  • 关系运算符 > >= < <=

  • 相等运算符 == != === !==

  • 逻辑运算符 先&& 后||
